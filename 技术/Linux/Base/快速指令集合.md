#### 用户

```shell
#修改用户UID
usermod -u NEW_UID USERNAME
#修改文件属性
chown [选项] 新用户:新组 文件夹名
# 将folder文件夹的用户改为user1，组改为group1
chown user1:group1 folder
# 递归更改目录及其所有内容
chown -R user1:group1 folder
# 只更改文件夹的用户归属
chown user1 folder
chown -R user1 folder
```

#### 组

```shell
#用户所在的所有组
groups USERNAME
# 使用chown修改文件权限组
chown :group1 folder
chown -R :group1 folder
# 或使用chgrp命令修改文件权限组
chgrp group1 folder
chgrp -R group1 folder
```

#### 软链接

```shell
#创建文件软链接	
ln -s 源文件 链接名	
ln -s /path/to/file.txt my_link
#创建目录软链接	
ln -s 源目录 链接名	
ln -s /path/to/dir my_dir_link
#删除软链接	
rm 链接名	
rm my_link
#删除软链接	
unlink 链接名	
unlink my_link
```
